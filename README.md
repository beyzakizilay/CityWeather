# City Weather Web Uygulaması

## Proje | PYTHON & Django

### Proje Açıklaması
<p>City Weather, kullanıcıların Türkiye’deki şehirlerin hava durumlarını görebileceği ve hızlı bir şekilde şehirlere ulaşabileceği bir Django tabanlı web uygulamasıdır.Uygulama, basit ama modern bir arayüze sahiptir ve kullanıcı dostudur.<p>

---

## Giriş / Kayıt Ekranı
<p>Kullanıcı adı ve şifrenizi girerek sisteme giriş yapabilirsiniz.Kayıtlı değilseniz kullanıcı adı ve şifre yazarak kayıt olabilirsiniz.<p>
